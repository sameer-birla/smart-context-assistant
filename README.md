# Smart Context Assistant

## Problem
Students struggle with planning, exam pressure, and productivity.

##  Solution
A smart assistant that provides context-aware suggestions based on user input.

##  Features
- Context-aware responses
- Time-based suggestions
- Quick suggestion buttons
- User history tracking
- Firebase integration (real-time data storage)

##  Tech Stack
- HTML
- CSS
- JavaScript
- Firebase Firestore

##  Live Project
https://smart-context-assistant.web.app

##  How it Works
User input → logic processing → response → stored in Firebase
