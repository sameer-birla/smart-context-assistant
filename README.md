# Smart Context Assistant

 Live Demo: https://smart-context-assistant.web.app  
 GitHub Repo: https://github.com/sameer-birla/smart-context-assistant  

---

##  Problem Statement
Students often struggle with exam planning, productivity, and stress management.

---

##  Solution
This Smart Context Assistant provides context-aware suggestions based on user input like exams, study, and productivity.

---

##  Features
- Context-aware responses
- Time-based suggestions (morning/evening logic)
- Quick action buttons (instant queries)
- History tracking (localStorage)
- Firebase Firestore integration (stores user queries)

---

##  Tech Stack
- HTML
- CSS
- JavaScript
- Firebase (Firestore + Hosting)

---

##  How It Works
User enters a query →  
System processes keywords →  
Generates smart response →  
Stores interaction in Firebase  

---

##  Google Services Used
- Firebase Firestore → stores user queries
- Firebase Hosting → deploys live project

---

## Output
Simple interactive UI that responds to user queries in real-time.

---

##  Future Improvements
- Add AI-based responses
- User authentication
